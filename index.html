Projects

Summer 2017 has really been productive for me. I started as an web developer intern at SBCotton and really improved my skills of programming. I learned new programming languages such as JavaScript, NodeJS, and jQuerry. SBCotton had several projects going on that dealt with Web Application Development for agricultural based companies. As a part of SBCotton Intern team, I worked on the project with Smart Notification System (SNS) with Amazon Web Service. Using NodeJS, the SNS service was tasked to send SMS to cell phones in Amazon Web Service. The code was executed using Lambda function in AWS. Part of the project was to learn the modules used by NodeJS such as Bluebird. Promises were used in the code to avoid the issues of "Callback Hell".
